Hero Carousel Background Images
================================

This folder contains background images for the hero carousel slides.

Required Images:
1. coffee-beans.jpg - Coffee beans close-up (Slide 1: Premium Coffee)
2. farmers.jpg - Farmers working in coffee fields (Slide 2: Empowering Youth)
3. organic-farm.jpg - Organic coffee plantation (Slide 3: 100% Organic)

Recommended Specifications:
- Resolution: 1920x1080 or higher
- Format: JPG or WebP for better performance
- File Size: Under 500KB (optimized for web)
- Aspect Ratio: 16:9

Tips:
- Use high-quality, professional images
- Ensure good contrast for text readability
- Images should align with the slide's message
- Consider using free stock photos from:
  * Unsplash.com
  * Pexels.com
  * Pixabay.com

Note: The carousel has a dark gradient overlay (green to coffee brown) 
to ensure text remains visible over any background image.
